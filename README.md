# Vertx-Dashboard

Built with React. Written in JavaScript, Tailwind CSS.

## 🔧 Technologies Used

- JavaScript
- React
- Tailwind CSS
- ChartJs

## 📦 Installation

```sh
git clone https://github.com/AshwiniParaye1/Vertx-Dashboard.git
cd Vertx-Dashboard
npm install
```
